학습 루프를 데이터로 남긴다.

Timer

포모도로 세션을 기록하고 커스텀 목표/휴식 설정을 관리합니다.

Analytics

대시보드, 위클리 뷰, 캘린더 히트맵으로 학습 패턴을 확인합니다.

AI chat

학습 문서와 코드 파일을 파싱해 chat context로 활용하는 실험 surface입니다.

Next.js App Router와 Supabase.

Frontend

Next.js, React 19, Tailwind, Zustand, Recharts 기반 UI입니다.

Backend

Supabase Auth, PostgreSQL migrations, API routes, server/client helpers를 사용합니다.

Deploy
